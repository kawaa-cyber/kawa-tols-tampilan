## ⚡ **CARA INSTALL**

### **Method 1: Auto Install (Recommended)**
```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/kawacyber/kawa-terminal
cd kawa-terminal
python install.py
```

### **Method 2: One-Liner Install**
```bash
curl -sL https://raw.githubusercontent.com/kawacyber/kawa-terminal/main/setup.sh | bash
```

### **Method 3: Manual Install**
```bash
# Step 1: Update Termux
pkg update && pkg upgrade -y

# Step 2: Install dependencies
pkg install python python2 git nano wget curl -y

# Step 3: Clone repository
git clone https://github.com/kawacyber/kawa-terminal
cd kawa-terminal

# Step 4: Install Python modules
pip install requests colorama tqdm psutil

# Step 5: Run
python kawa_cyber.py
```

### **Method 4: Download ZIP**
1. Download ZIP dari GitHub
2. Extract ke folder `~/kawa-terminal`
3. Jalankan `python install.py`

## ⚙️ **KONFIGURASI**

Edit file `config.json` untuk kustomisasi:

```json
{
    "theme": "matrix",
    "auto_update": true,
    "matrix_speed": 0.05,
    "language": "en",
    "animations": true,
    "show_banner": true,
    "tools_directory": "/sdcard/kawa-tools",
    "last_update": "2026-03-15",
    "version": "3.0"
}
```

### **Penjelasan Config:**
- `theme` - Theme default (matrix/neon/dark/cyber/blood/royal)
- `auto_update` - Auto update tools (true/false)
- `matrix_speed` - Kecepatan matrix rain (0.01 - 0.1)
- `language` - Bahasa (en/id)
- `animations` - Aktifkan animasi (true/false)
- `show_banner` - Tampilkan banner (true/false)

## 🐛 **TROUBLESHOOTING**

### **Error: Permission denied**
```bash
chmod +x kawa_cyber.py
python kawa_cyber.py
```

### **Error: Module not found**
```bash
pip install -r requirements.txt
```

### **Error: Storage access**
```bash
termux-setup-storage
# Allow permission when prompted
```

### **Error: Python version**
```bash
pkg install python@latest
```

### **Error: Git not found**
```bash
pkg install git -y
```

### **Error: Can't connect to GitHub**
```bash
# Check internet connection
ping google.com

# Use HTTP instead of HTTPS
git config --global url."http://github.com/".insteadOf https://github.com/
```
